About the game

A truth was sealed away.
You just found a piece of it.

Silent Horizon is a top-down action-adventure game set in a world that has been deliberately fractured. Ancient knowledge was shattered into Fragments and scattered across a dangerous, ever-shifting map. No one alive knows why — or who did it.

You begin with nothing. A sword, an open world, and the creeping sense that the landscape around you is hiding something. Explore, fight, and survive long enough to find the first Fragment. What it shows you will change everything.

"He no longer knows what he protects — only that he must."

The world is guarded. A Knight — bound to an ancient seal — stands between you and the truth. His shield faces every threat. Strike him from the front and your sword will glance off gold. Circle him. Find his blind side. Wear him down.

This is only the first Fragment. The horizon goes further than you think.

Getting started

How to install & run

Windows
  1. Click the Windows download button above
  2. Once downloaded, right-click the .zip file and select Extract All
  3. Open the extracted folder
  4. Double-click SilentHorizon.exe to launch the game
If Windows Defender warns you, click More info → Run anyway. This is normal for unsigned indie games.
Linux
  1. Click the Linux download button above
  2. Extract the .zip file — right-click and Extract Here, or run unzip SilentHorizon-Linux.zip
  3. Open a terminal inside the extracted folder
  4. Make the launcher executable: chmod +x SilentHorizon
  5. Run the game: ./SilentHorizon
Requires Java to be installed. Most Linux distros include it — if not, run sudo apt install default-jre
macOS  COMING SOON
  1. Download the correct version for your Mac — Apple Silicon (M1, M2, M3, M4 chips) or Intel (older Macs)
  2. Extract the .zip — double-click it in Finder
  3. Drag the .app file to your Applications folder
  4. The first time you open it, macOS will block it. Go to System Settings → Privacy & Security and click Open Anyway
macOS blocks apps from unidentified developers by default. This is expected for indie games — the game is safe to run.

Changelog

  • First public release — Fragment I complete

  • Melee and archer enemy types with distinct AI behaviour

  • Initial prototype — movement, camera, basic tile map
  • Sword combat and enemy pathfinding
  • Sword arc glitching when player changed direction mid-swing — fixed with direction lock on swing start
  • Player unable to move after respawn — fixed with held-key state polling
